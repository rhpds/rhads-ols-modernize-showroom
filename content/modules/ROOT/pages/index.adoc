= Accelerating Application Modernization With AI: Red Hat Developer Lightspeed

Welcome to the AI-powered application modernization workshop!

== What you'll learn

In this workshop, you will:

* Analyze legacy applications using Migration Toolkit for Applications (MTA) with AI-powered risk assessment
* Generate modernized code using Red Hat Developer Lightspeed with AI assistance
* Automate collection of best practices and company policies for code generation
* Deploy modernized applications to Red Hat OpenShift

== Who this is for

This workshop is designed for developers, architects, and platform engineers who want to learn how to migrate legacy applications to OpenShift using AI-infused Developer Lightspeed for MTA.

**Experience level**: Intermediate

== Prerequisites

Before starting this workshop, you should have:

* Basic understanding of Java application development
* Familiarity with containerization concepts
* Basic knowledge of OpenShift or Kubernetes
* Experience with Git and version control
