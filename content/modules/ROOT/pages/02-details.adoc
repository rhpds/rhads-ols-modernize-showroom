== Module breakdown

=== Module 1: Assessment and analysis (60-90 minutes)

In this module you will:

* Explore the MTA application inventory
* Learn about application profiles, tags, and archetypes
* Complete an assessment questionnaire for an archetype
* Analyze application source code using MTA
* Review analysis reports including effort estimates, issues, and dependencies

=== Module 2: Refactor and deploy (60-90 minutes)

In this module you will:

* Identify and fix migration issues in source code
* Refactor legacy configuration code to use Spring standards
* Update database connection properties
* Re-run MTA analysis to verify fixes
* Build and deploy the modernized application to OpenShift
* Test the deployed application
